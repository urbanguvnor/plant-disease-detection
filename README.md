## Project Overview
This project implements a Convolutional Neural Network (CNN) to automatically detect and classify plant diseases from leaf images using the PlantVillage dataset.

## Dataset
- **Source:** PlantVillage Dataset (Kaggle)
- **Size:** 54,305 images
- **Classes:** 38 disease categories
- **Plants:** 14 crop species

## Methodology
- **Model:** Convolutional Neural Network (CNN)
- **Framework:** TensorFlow/Keras
- **Platform:** Google Colaboratory
- **GPU:** NVIDIA T4 (via Google Colab)
